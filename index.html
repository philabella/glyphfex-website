Frequently asked questions

A .glyphfex file is a single SQLite database that contains all your entries, tags, pipeline configurations, attachments, and audit history. SQLite is the same database engine trusted by billions of devices worldwide — it powers apps on every smartphone, browser, and operating system. You can copy it, back it up to a USB drive, email it to a colleague, or store it on a network share. No server required. Local-first architecture means your data stays fast, no matter how many entries you have.

No. GlyphFex works 100% offline. The only time you need internet is to download the installer and activate your license. After that, everything runs locally on your machine.

GlyphFex runs on Windows 10 (version 1903 or later) and Windows 11. Both x64 and ARM64 are supported. Mac and Linux versions are not currently available.

Team mode uses SQL Server Express (free from Microsoft) running on your local network. No cloud, no subscription to Microsoft. GlyphFex connects via Windows Authentication — no extra passwords. Your IT person (or you) runs a one-time SQL Server setup, then everyone on the LAN connects automatically.

Yes. GlyphFex imports CSV files with automatic field mapping. Export your Excel tracker as CSV, import into GlyphFex, and your entries are created with tags, custom fields, and pipeline stages mapped automatically.

GlyphFex records a full audit trail for every change — who changed what, when, and what the previous value was. This provides the traceability that ISO 9001 and AS9100 auditors look for. You can export the audit history as PDF or CSV for compliance documentation.

Because visibility shouldn't have a price tag. Put GlyphFex on every shop floor PC, every TV display, every manager's desk. The more people who can SEE what's going on, the faster your shop runs. You only pay for people who need to CREATE or EDIT entries.

GlyphFex keeps automatic rotating backups (.bak, .bak.1, .bak.2) every time you save. You can also use File → Backup Database for a one-click manual backup (works for both SQLite and SQL Server). For full disaster recovery, use Export Full Project to create a ZIP including all attachments. A built-in Disaster Recovery Guide (Help menu) gives your IT team step-by-step restore instructions.

Your data is always yours. The .glyphfex file is a standard SQLite database. If you stop renewing, you keep read-only access to all your data and can export everything to CSV or PDF. We never hold your data hostage.
