Quick Start Guide

From download to managing your shop in 5 steps.

1

Install the MSI

Double-click the installer. Accept the defaults. GlyphFex will appear in your Start menu.

2

Create a new project

File → New Project. Choose where to save your .glyphfex file. Pick an industry template or start blank.

3

Configure your pipeline

The template sets up default stages. Customize them in Tools → Edit Pipeline. Add or remove stages to match your workflow.

4

Create your first entry

Click "New Entry" (Ctrl+Shift+N). Fill in the ref number, customer, and details. Save it. Your first job is being tracked.

5

Explore the dashboard

Switch to Dashboard (Ctrl+1) to see your entries in card, table, or timeline view. Apply filters, check analytics, and explore.

Video: Getting started in 5 minutes

Need multi-user? Scale when ready.

Start with the solo .glyphfex file. When your team grows, use the built-in migration wizard to move to SQL Server Express (free from Microsoft). Same interface, shared across your whole team on your local network.
