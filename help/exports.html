Exports & Reports

Export your data as PDF, CSV, Excel, or QuickBooks format. Import entries from CSV files.

PDF Entry Export

Print any individual entry as a detailed, professionally formatted PDF document. The PDF includes:

PDF Entry Export Screenshot of a printed PDF entry showing all fields, stage timeline, and QR code

The PDF uses a professional compact layout with your company name in the header, a prominent reference number, the entry's Smart Description (if configured), and a clean 3-row job details section with tight 20mm margins for maximum content per page.

  • Header — Company name, reference ID (large 18pt), Smart Description (11pt), project name, and generation date
  • All fields — Built-in fields, custom fields, tags, and comments
  • Stage timeline — Visual history of every pipeline stage transition with timestamps and durations
  • Job costing — Unit price, quantity, and total value (when applicable)
  • Part history — If the entry has a part number, a table of other entries sharing the same part number (capped at 25)
  • QR code — A scannable glyphfex://entry/{id} link that opens the entry directly when scanned with a barcode reader

Access: Right-click any entry on the Dashboard and select Print, or open the Entry Detail Window and click Print.

Tip The PDF entry export is ideal for creating physical job travelers, customer-facing summaries, or archival records. Print one for every job and attach it to the physical work piece as it moves through the shop.

PDF Dashboard Report

Generate a summary report of the entries currently visible on your Dashboard. The report includes:

  • Filter description — Which filters were active when the report was generated
  • Entry table — A tabular listing of all matching entries with key columns
  • Statistics summary — Total entries, status breakdown, and other aggregate data

Access: Click the Export PDF button in the Dashboard toolbar. The report respects your current filters, search query, and sort order.

Job Traveler PDF

A Job Traveler is a single-page PDF designed to accompany a physical job through the shop. It includes:

Job Traveler Screenshot of the Job Traveler PDF showing the single-page layout with large QR code, ref number, and key fields
  • Reference ID and job description
  • Customer name, PO number, due date, and other key fields
  • Pipeline stages as a visual checklist
  • A large QR code that links to glyphfex://entry/{id}

Scan the QR code with any standard barcode scanner connected to a PC running GlyphFex. The entry opens instantly — no searching, no typing. This is especially powerful on the Shop Floor kiosk where operators can scan a QR code to clock in/out of a specific job.

Tip Print Job Travelers on durable paper or laminate them for shop floor use. The QR code works even when printed in black and white.

CSV Export

Export your entries to a standard CSV (comma-separated values) file that can be opened in any spreadsheet application.

Shortcut: Ctrl+E or File > Export > CSV

The CSV export respects your current Dashboard filters — only the entries matching your active filters are exported. This lets you export a subset of data (e.g., "all entries in Fabrication stage" or "all Rush priority jobs") without exporting everything.

Note GlyphFex includes formula injection defense in CSV exports. Cells that begin with =, +, -, or @ are prefixed with a single quote (') to prevent spreadsheet applications from executing them as formulas. This protects against a known CSV injection attack vector.

Excel Export

Export your data as a formatted .xlsx Excel file. The Excel export includes proper column headers, data types, and basic formatting for easy use in Microsoft Excel, Google Sheets, or LibreOffice Calc.

Access: File > Export > Excel

QuickBooks Format

Export entries in a CSV format compatible with QuickBooks invoice import. This is useful for accounting integration when you need to transfer job data (reference IDs, customer names, quantities, prices) into your accounting system.

Access: File > Export > QuickBooks CSV

Tip The QuickBooks export maps GlyphFex fields to QuickBooks invoice columns. Review the mapping before importing to ensure field alignment matches your QuickBooks configuration.

Full Project ZIP

Export your entire project as a ZIP archive containing multiple CSV files for comprehensive backup or data portability.

Shortcut: Ctrl+Shift+E or File > Export > Full Project

The ZIP archive contains:

  • entries.csv — All entries with all fields
  • audit.csv — Complete audit trail history
  • categories.csv — Category and tag configuration
Tip Use the Full Project ZIP as a periodic backup or when you need to migrate data to another system. The CSV files are human-readable and can be processed by any data tool.

CSV Import Wizard

Import entries from an external CSV file using the 3-step import wizard.

Shortcut: Ctrl+I or File > Import from CSV

CSV Import Wizard Screenshot of the CSV Import wizard showing the column mapping step with source CSV columns mapped to GlyphFex fields
  1. Select file — Browse to your CSV file. GlyphFex reads the header row and shows a preview of the data.
  2. Map columns — Match each CSV column to a GlyphFex field. The wizard suggests mappings based on column names, but you can adjust them manually. Unmapped columns are skipped.
  3. Import — GlyphFex validates the data and imports matching rows. A summary shows how many entries were created, skipped, or errored.

Duplicate handling

When an imported entry has the same Reference ID as an existing entry, you can choose:

  • Skip — Do not import the duplicate (keep existing)
  • Overwrite — Replace the existing entry with the imported data
  • Create new — Import as a separate entry (appends a suffix to avoid ID collision)
Warning The import wizard validates data before committing. If required fields are missing or validation patterns do not match, those rows are flagged and skipped. Review the import summary carefully.

Built-In PDF Guides

GlyphFex includes three professionally formatted PDF guides that are generated on demand and opened in your default PDF viewer:

Getting Started Guide (F1)

A comprehensive 23-page guide covering installation, first launch, project creation, entry forms, pipeline configuration, Dashboard views, Shop Floor kiosk, settings, and FAQ. Access from Help > Getting Started Guide or press F1.

Process Guide

An 8-page guide focused on workflow processes: how to use pipeline stages, manage entries through their lifecycle, handle multiple pipelines, use the Shop Floor kiosk, and track SLA compliance. Access from Help > Process Guide.

Feature Overview

An 11-page marketing-ready PDF listing 40+ features across 9 categories, each with a description and a "How a shop uses this" use case. Includes a pricing page with all 4 editions. Access from Help > Feature Overview.

Tip The Getting Started Guide is a great resource to share with new team members. Print it or email the PDF so they can learn GlyphFex at their own pace.

Timesheet Reports

Generate timesheet summaries from clock in/out records captured on the Shop Floor page.

Access: Tools > Timesheet Report

  • Filter by date range — Select a start and end date to scope the report
  • Filter by worker — Choose a specific worker or view all workers. Workers are populated from clock in/out records within the selected date range.
  • Summary view — Shows total hours per worker, per entry, with clock-in and clock-out timestamps
  • Export to PDF — Generate a printable timesheet PDF for payroll, billing, or record-keeping
Note Timesheet data comes from the Shop Floor's clock in/out feature. If your team uses the Shop Floor kiosk for time tracking, this report is the fastest way to extract that data for payroll.
